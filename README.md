# python-pdf

Important notes about libraries for data extraction, analysis,
conversion & manipulation of PDF (and other) documents:
- [`pypdf`][1],
- [`PyMuPDF`][2],
- [`tabula-py`][3].

[1]: https://pypdf.readthedocs.io/en/latest/index.html
[2]: https://pymupdf.readthedocs.io/en/latest/
[3]: https://tabula-py.readthedocs.io/en/latest/

## Installation

### `pypdf`

Follow: https://pypdf.readthedocs.io/en/latest/user/installation.html#installation

```bash
pdm add pypdf[full]
```

### `PyMuPDF`

Follow: https://pymupdf.readthedocs.io/en/latest/installation.html#id1

```bash
pdm add pymupdf
```

### `tabula-py`

Follow: https://tabula-py.readthedocs.io/en/latest/getting_started.html#installation

```bash
pdm add tabula-py[jpype]
```
